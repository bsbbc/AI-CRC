# AI-CRC
### raspberry-pi robot-cleaner algorithm
--------------------------------------
This python file is a robot-cleaner algorithm based on Grid Map and Exhaustive Search

이 파이썬 파일은 격자무늬 맵과 완전탐색을 기반으로 한 로봇청소기 알고리즘입니다.

--------------------------------------
